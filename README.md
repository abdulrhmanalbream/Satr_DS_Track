# Satr_DS_Track

In-Depth Learning Formulas.
  
The track included several areas such as:
-Python
-SQL
-Git
- Analyze and process data via libraries: NumPy, Pandas
- Data representation via libraries: Matplotlib, Seaborn
- EDA exploratory data analysis
- Machine Learning (ML)
